**Clatter Code** is the ultimate **Agentic Logic Orchestration System**. It is a "Combo" platform that seamlessly integrates the autonomous reasoning of **OpenManus**, the interactive terminal structure of **OpenCode**, and the high-performance execution of **Rust** (inspired by **Bun**).

---

## 🌪 The Orchestration System

Clatter Code doesn't just run commands; it orchestrates a multi-phase workflow:
1. **Strategic Planning:** Decomposes complex tasks into actionable steps (OpenManus Logic).
2. **Autonomous Execution:** Executes shell commands, modifies files, and debugs in real-time.
3. **Performance Optimization:** Uses a dedicated Rust engine for lightning-fast file operations (Bun Philosophy).

---

## 🔌 Multi-Provider Support

Clatter Code connects to any AI brain, local or cloud:
- **Cloud:** OpenAI, Anthropic, Google Gemini, xAI.
- **Local:** Ollama, LM Studio.

## 🚀 One-Liner Combo Installation

Get the entire ecosystem up and running with a single command:
```bash
git clone https://github.com/mohamed-dev-labs/clatter-code.git && cd clatter-code && chmod +x install.sh && ./install.sh
```

## 🛠 Usage Guide

### 1. The Orchestrator (CLI Mode)
Run complex tasks with your preferred provider:
```bash
clatter run "Build a full-stack blog with Next.js and Prisma" --provider anthropic
```

### 2. The Graphical TUI
Launch the unified dashboard to monitor the agent's logic:
```bash
clatter
```

### 3. Clatter Package Manager (CPM)
Manage skills and plugins just like `npm` or `npx`:
```bash
# Install a new skill
clatter install @skill/web-search

# Execute a skill directly
clatter x @skill/web-search "Search for latest AI news"

# List installed skills
clatter pkgs
```

### 4. Automated Setup
Configure your environment in seconds:
```bash
clatter setup
```
